js的基本语法

1.语句

js程序的执行单位为行，也即是一行一行地执行，一般情况下，每一行就是一个语句。

2.变量

2.1概念

变量就是为值取名，引用这个名字，就等同于引用这个值。

js是一种动态类型的语言，变量的类型没有限制，可以随时更改类型。

变量声明并赋值两次，第二次赋值会覆盖掉第一次赋值。

2.2变量提升

js引擎工作方式，先解析代码，然后获取所有被声明的变量，然后再一行一行的运行，所有变量的声明语句，都会被提升到代码的头部

3.标识符

命名区分大小写，首字母不能是数字，不能包含星号，加减号等，不能使用保留字等等

4.注释

单行注释//

多行注释/* */

由于历史原因<!- -和- ->也被视为合法的单行注释，但是- ->必须放在词首时才会起作用

5.区块

js使用大括号将多个相关语句组合在一起，称为“区块”

6.条件语句

7.循环

循环语句用于重复执行某个操作，有多种形式。

7.1while循环

7.2for循环

7.3do…while循环

7.4break语句和continue

都具有跳转作用，可以让代码不按既有顺序执行

7.5标签(label)

标签可以时任意的标识符，通常与break和continue语句配合使用
